For UI Designer artifacts, the Bonita REST API resources are exported in the page.properties files. Those resources are used at deployment to grant permission to those REST API to users with access to this page.

Manual detection

You can manually declare the Bonita REST API resources which you're using in your artifact when the automatic detection fails to found them. You need to select an HTTP Verb, and type/choose the REST API on the value field. This Bonita resource will export in page.properties files.

  You should always declare the resource on the most atomic artifact (in fragment, if the resource is used in a fragment, in a custom widget, or at page level)
Automatic detection

When you drag and drop a widget or a fragment, some Bonita resources are automatically found and added in the Bonita resource table. The scopes column of the Bonita resources table helps you identify what artifact declare a given resource.
