Use this container to create tabs which can display content on the page, form, or layout.

It's possible to display tabs vertically or horizontally.

To Display the tab vertically and on the left of the content, you can read this how to learn more about customization

Display type allow you to customize format type of tabs. It can be tabs style or pills style.

If you want more options for each tab, refer to Tab properties usage.

The Tab's properties allows you to customize default display and behaviour of tabs based on data and/or JS expressions.

  • Title: it will be the tab title displayed. It can be calculated dynamically to display and render html.
  • hidden (Defaults: false): Whether tab is hidden or not.
  • disabled (Defaults: false): Whether tab is clickable and can be activated.
  • cssClasses (Defaults: ""): Add css classes on tab. This css class will be attached to tab-heading html element.
